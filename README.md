üìå Project Overview

OpenCatalog is a modern web application built with Next.js, TypeScript, and Tailwind CSS.
It allows users to browse, search, and explore data fetched from a public API in a clean, responsive interface.

Purpose: 

To demonstrate proficiency in Next.js App Router, React components, and API integration while maintaining a professional, user-friendly UI.

Features:

Home page with a grid/list of items from the API

Dynamic details page for each item (/item/[id])

Search and optional filter functionality

Loading skeletons and error handling

Mobile-first responsive design
